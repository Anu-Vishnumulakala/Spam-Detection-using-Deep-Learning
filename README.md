# Spam-Detection-using-Deep-Learning
Built an end-to-end spam email classifier using deep learning 
## Overview
- **Problem:** Classify email text as **Spam** or **Not Spam**
- **Model:** Deep Learning
- **Frontend:** Streamlit
- **Dataset:** SMS Spam Collection Dataset
- **Key Tools:** TensorFlow/Keras, Streamlit, Tokenizer, Pickle
## Evaluation Summary
- Used **confusion matrix** to assess performance
- Model was strong at detecting **Not Spam** but weaker at detecting **Spam**
- True Positives (Spam correctly identified): **88.11%**
- False Negatives (Missed Spam): **20%**
